Debian 1.1 (Buzz)


Buzz est le nom de code d'une ancienne distribution Debian Stable. Elle a été publiée le 17 juin 1996 sous le nom de Debian GNU/Linux 1.1. C'était la première vesion de Debain avec un nom de code. Ce nom vient, comme tous les autres jusqu'ici, de celui d'un personnage du film d'animation Toy Story. Dans ce cas, Buzz l'Eclair.


A cette époque, Bruce Perens avait pris la tête du projet après Ian Murdock, et il travaillait chez Pixar, la compagnie qui avait produit le film. Cette version était totalement en ELF (Executable and Linkable Format), utilisait le noyau Linux 2.0, elle contenait 474 paquets. Elle est maintenant obsolète et ne bénéficie plus des mises à jour de sécurité. La version suivante est Rex.
